# Heterogeneous-Parallelization
# практики и асайменты

репозиторий с заданиями по параллельному программированию и всякому разному

## структура

каждая практика в отдельной папке со своим readme и кодом

## что внутри

- задания с openmp
- работа с параллельными алгоритмами
- сравнение производительности
- всякие эксперименты с потоками

## требования

зависит от конкретного задания, но в основном:
- компилятор с поддержкой c++11 или новее
- openmp (идёт с gcc/g++)
- иногда специфичные библиотеки (будет указано в папке)
